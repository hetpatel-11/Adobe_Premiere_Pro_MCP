# Claude Code 편집 요청 프롬프트 템플릿
# (유튜브 영상 제작 및 편집 만능 템플릿 통합 버전)

---

## ⚠️ 0단계 — 시작 전 필수 (EDITING_WORKFLOW.md 준수)

**컷편집을 시작하기 전에 반드시:**
1. `batch_transcribe.py` 로 **모든 클립** 전사 → `dialogue_map.json`
2. `srt_export.py` 로 SRT 생성 → `create_caption_track` 으로 시퀀스에 자막 삽입
   → `read_sequence_captions` 로 삽입 검증
3. **이해도 게이트 통과**: 클립별 1줄 요약 + 전체 스토리 흐름 + 하이라이트 후보 +
   목표 길이(10~20분) 계획을 사용자에게 제출하고 "진행" 확인을 받는다.
4. 확인 후에만 컷편집 실행.

> 자막 없이 프레임만 보고 편집 금지. 대화·감정·이야기가 브이로그의 본체다.

## 막시마 김밥 영상 편집 요청 (즉시 사용 가능)

```
pipeline/output/master_edit_plan.json 을 읽어줘.
그리고 아래 조건으로 현재 Premiere Pro 시퀀스를 컷편집 해줘.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[영상 정보]
- 주제: 한국에 사는 독일 19살 여성 막시마가 김밥을 만드는 과정
- 등장인물: 막시마(독일), 촬영자(한국인)
- 촬영 장소: 한국 슈퍼마켓 + 실내 주방
- 톤: 재미있고 엉뚱한 예능형 리포트 스타일
- 목표 길이: 12~15분

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[타임라인 구조 - 마스터 템플릿 적용]

▶ 00:00~01:30 | 오프닝 & 훅 (Hook)
- 막시마의 가장 황당하거나 웃긴 순간을 3~5초 선공개 (시청 이탈 방지)
- 막시마 소개 + 오늘의 미션(김밥 만들기) 직관적 제시
- 보이스오버(VO) 자리 확보 (1~2초 여백)
- 편집 기술: Punch-In (scale 125%), 인트로 페이드인

▶ 01:30~05:00 | 빌드업 (Build-up)
- 막시마와의 첫 만남, 아이스브레이킹
- 슈퍼마켓 이동 및 재료 구매 과정
- 단순 반복 구간(걷기, 이동)은 Speed Ramping으로 압축
- 편집 기술: Speed Ramping, Split Screen (막시마 표정 vs 재료 클로즈업)

▶ 05:00~09:00 | 딥 다이브 (Deep Dive)
- 주방에서 재료 손질하며 나누는 진솔한 대화
- 막시마의 한국 정착기, 독일과의 문화 차이 토크
- 자막 톤: 따뜻하고 정중하게
- 편집 기술: J-Cut / L-Cut (대화 흐름 유지)

▶ 05:00~06:00 구간 | ★핵심 반전 배치 (기승전결 '전')
- 훈훈한 흐름을 깨는 위기/실수 장면 필수 배치
  예: 재료 실수, 예상 밖 반응, 당황하는 막시마
- Off-beat Mute 적용 (BGM 칼같이 음소거 → 정적 효과)

▶ 09:00~12:00 | 클라이맥스 (Climax)
- 김밥 완성 및 생애 첫 직접 만든 김밥 시식
- 막시마의 최고조 리액션 포착
- 편집 기술: Slow Zoom-In (완성 김밥 클로즈업), Off-beat Sound Cut

▶ 12:00~13:30 | 아웃로 (Outro)
- 오늘 소감 나누기, 막시마 응원 메시지
- 쿠키 영상 (NG or 비하인드) 배치
- 편집 기술: Audio Ducking (보이스오버 구간 BGM 자동 다운)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[스토리텔링 - 옵션 B: 현대식 기승전결 적용]

기(起): 막시마의 유쾌한 첫 등장 + 오늘 미션 제시
승(承): 슈퍼마켓 → 주방 → 재료 손질 순탄하게 진행
전(轉): 5~6분 지점 위기/반전 (실수, 황당한 순간)  ← 필수
결(結): 위기 극복 후 김밥 완성 + 시식 카타르시스

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[편집 기술 체크리스트 - MCP 실행 항목]

□ Punch-In
  → 막시마의 황당한 표정/리액션 순간
  → set_clip_properties: scale 100% → 125%, 순간 적용

□ Speed Ramping
  → 걷기/이동/단순 칼질 구간
  → speed_change: 구간별 200~400% 가속

□ Split Screen
  → 막시마 표정 vs 재료/레시피 화면 동시 표현
  → Video Track 2 사용 + set_clip_properties (위치/크기 조정)

□ J-Cut / L-Cut
  → 대화 장면 전환 시 오디오 먼저 전환
  → trim_clip으로 오디오/비디오 오프셋 적용

□ Off-beat Mute
  → 결정적 실수/황당한 장면
  → set_audio_clip_disabled 또는 오디오 레벨 0

□ Slow Zoom-In
  → 완성된 김밥 클로즈업, 감정 피크 순간
  → add_keyframe: scale 100%→115% 2~3초에 걸쳐 서서히

□ Audio Ducking
  → 보이스오버 구간
  → setup_ducking: VO 트랙 기준 BGM -20dB 자동 감소

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[Claude 실행 순서]

1. master_edit_plan.json 읽기
2. 각 장면 full_text + frame_path 분석
3. 위 타임라인 구조에 맞게 장면 배치 계획 수립
4. export_frame MCP로 핵심 장면 시각 확인
5. 장면별 편집 기술 결정
6. MCP 순차 실행:
   trim_clip → add_to_timeline → set_clip_properties
   → speed_change → add_keyframe → add_transition
   → setup_ducking → add_text_overlay
```
